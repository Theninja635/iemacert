# IEMA Cert

Responsive static recreation of the IEMA Cert website, built with HTML, CSS, and JavaScript.

## Local preview

Open `index.html` directly in a browser, or use any static web server from the project folder.

## Manual deployment

This is a static website with no required build or deployment process. Upload the contents of this folder to your chosen provider (such as Vercel, Netlify, or regular web hosting), keeping the folders and files together.

## Project structure

```text
.
├── index.html
├── aboutus.html
├── ourservices.html
├── contactus.html
├── css/style.css
├── js/main.js
├── images/                 # Replace visual placeholders with approved assets
├── assets/                 # Reserved for approved external assets
```
